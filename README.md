# El Paso Zoning Code Ingestion & State Compliance Audit Tool

This repository provides a data pipeline designed to pull the actual text of **Title 20 (Zoning Ordinance)** for the City of El Paso, Texas, directly from the underlying unauthenticated REST API endpoints utilized by Municode. 

The goal of this tool is to systematically aid urban planners and policymakers in checking municipal text alignment with changing state-level housing legislation (e.g., Texas SB 15, SB 840).
